Keep users on your page, out of the administration area for login, profile updates, and registration

== Description ==

 Login Register Profile enables you to keep users out of the administration areas by moving the login page, register page, and profile page to the front area of the website, where it belongs.    
  
Features

*   Custom Field Support (stores them in usermeta)
*   WooCommerce integration  
    
*   JanRain integration
*   Customize the forms
*   Customize the pages
*   Visible Profiles for users including latest blog posts, comments, bio, and purchases

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
