== Description ==

A lightweight plugin that will add the submit buttons for Digg, Stumble, Del.icio.us, Slashdot, Twitter, Mixx, Dzone, Sphinn, Google, and Technorati to the bottom of posts. It can be configured to only show some of those options or all of them.


== Installation ==

1. Upload the spreadx directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under "Tools" in the Administration area, you can make changes to how the plugin displays the buttons.
4. If you want to change the images used, feel free.  They are located in the /wp-content/plugins/spreadx/images directory.  Just use the same image name, and it will change automatically.
