A very easy way to get your site onto Digg, Stumble, Del.icio.us, Slashdot, and Technorati.

== Description ==

A lightweight plugin that will add the submit buttons for Digg, Stumble, Del.icio.us, Slashdot, and Technorati to the bottom of posts. It can be configured to only show some of those options.


== Installation ==

1. Upload the spreadx directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under "Tools" in the Administration area, you can make changes to how the plugin displays the buttons.
