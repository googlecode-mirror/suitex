An easy to use script that allows you to create user groups and then manage their acess to specific pages within your website.

== Description ==
USecureX allows for the creation of user groups, then by adding users and pages to those groups, only those users can view that page.  There is no (realistic) limit to the number of groups you can create.



== Installation ==

1. Upload the usecurex directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under "Tools" in the Administration area, be sure to add a default page in which to redirect users to upon a failed link.
